Introduction

Certains caractères typographiques posent réellement problème quand on utilise un clavier AZERTY. Les caractères en question n'apparaissent pas toujours sur le clavier et la façon d'y accéder dépend beaucoup du système d'exploitation. Êtes-vous prêt à relever le défi ?

Il y a évidemment de nombreux symboles que vous pourriez être amené(e) à taper un jour ou l'autre, mais nous allons ici nous concentrer sur les principaux caractères particuliers qui feront de vous un ninja du clavier si vous savez les utiliser sans faire de copier-coller et sans passer par la fonction d'insertion ou de correction de votre logiciel de traitement de texte.

Voici les caractères abordés dans ce tutoriel : É È À Ç Ù œ æ Œ Æ « » – —
(Les deux derniers caractères correspondent respectivement au tiret demi-cadratin et cadratin)

Sous Linux

Le plus simple est que vous optiez pour la disposition « Français (variante) ». Vous aurez accès à tous les caractères avec les codes suivants :

  • É : Caps Lock + é
  • È : Caps Lock + è
  • À : Caps Lock + à
  • Ç : Caps Lock + ç
  • Ù : Caps Lock + ù
  • œ : Alt Gr + o
  • æ : Alt Gr + a
  • Œ : Shift + Alt Gr + o
  • Æ : Shift + Alt Gr + a
  • « : Alt Gr + w
  • » : Alt Gr + x
  • – : Shift + Alt Gr + (
  • — : Shift + Alt Gr + '

Sous Mac

  • É : Caps Lock + é
  • È : Caps Lock + è
  • À : Caps Lock + à
  • Ç : Alt + ç
  • Ù : Alt + ù
  • œ : Alt + q
  • æ : Alt + '
  • Œ : Alt + Shift + q
  • Æ : Alt + Shift + '
  • « : Alt + è
  • » : Alt + Shift + è
  • – : Alt + –
  • — : Alt + Shift + –

Sous Windows

  • É : Alt + 144
  • È : Alt + 212
  • À : Alt + 183
  • Ç : Alt + 128
  • Ù : Alt + 235
  • œ : Alt + 0156
  • æ : Alt + 145
  • Œ : Alt + 0140
  • Æ : Alt + 146
  • « : Alt + 0171
  • » : Alt + 0187
  • – : Alt + 0150
  • — : Alt + 0151
